# Task Management System using Python

## Project Description
This is a simple Task Management System developed using Python. It helps users to manage their daily tasks efficiently.

Users can:
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks

## Features
- Menu-driven program
- Easy to use
- Simple and clean code
- Runs in Google Colab or Python

## Technologies Used
- Python

## How to Run

1. Open Google Colab or Python IDE
2. Copy the code
3. Run the program
4. Follow the menu instructions
